# StopDuelBuffer
Auto-Decline World Buff / Flask Chugger Duels.

# Install
-> Click Clone or download
-> Download Zip
-> Drag Folder to World of Warcraft 1.12\Interface\AddOns
-> Rename it from StopDuelBuffer-Master to StopDuelBuffer
-> Activate ingame
