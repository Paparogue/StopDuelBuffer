# StopDuelBuffer
Auto-Decline World Buff / Flask Chugger Duels.

# Install
1.  Click Clone or download
2.  Download Zip
3.  Drag Folder inside StopDuelBuffer-Master to World of Warcraft 1.12\Interface\AddOns
4.  Activate it in Character selection
